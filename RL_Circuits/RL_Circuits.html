1 Introduction

The above figure shows a RL series circuit consisting of an inductor of inductance L connected in series with a resistor of resistance R. The switch, S, is closed at a time t=0 and remains closed.

9

Using Kirchhoff’s voltage law around the loop, we have

11

Vs​−I(t)⋅R−LdtdI(t)​=0

13-15

This differential equation can be solved for I(t), the current at any time. If the current at t=0 is zero, the solution is

17

I(t)=RVs​​(1−e−t/(L/R))

19-21

After a long time∗ (specifically, t≫τ), the current approaches a constant value of I=Vs​/R because the exponential term approaches zero. How quickly the exponential term approaches zero depends on a quantity called the RL time constant defined by

23

τ≡L/R

25

which has units of seconds when L is in Henrys (H) and R is in Ohms (Ω). Using τ, we have

27

I(t)=RVs​​(1−e−t/τ)

29-31

When t/τ≃0, e−t/τ≃e0=1, so I(t)≃(Vs​/R)(1−1)=0. As a result, we state that initially the inductor behaves like an open circuit because the current through it is nearly zero.

33

When t/τ≫1, the exponential term e−t/τ becomes much smaller than one, so I(t)≃(Vs​/R)(1−0)=Vs​/R. If we replace the inductor with a wire, this is the same current that we would find. As a result, we state that after a long time, an inductor behaves like a resistanceless wire.

35

∗ Informally, we often use phrases such as “after a long time”. This phrase is ambiguous because a reference length of time is not given. To be specific, one should instead state t≫tref​, where tref​ is a reference length of time.
